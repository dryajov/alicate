Input

new Input()

The Input class allows capturing data from an html element.

Version:
  • 1.0
Source:
  • components/input.js, line 13
Example
var someInput;
var inputModel = new Model({data: someInput});
var myInput = new Input({
     id: 'my-input',
     model: inputModel
});

var myLabel = new Label({
     id: 'my-label',
     model: inputModel
});
